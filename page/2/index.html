第一周学习笔记
这一周主要进行了以下几个方面的学习：
1）英语的持续性学习，每天背诵一些单词，听一些听力，练习自己的口语；
2）读了老师推荐的“Artificial Intelligence A Modern Approach”这本书，这周读的很慢，
了解了一些像“图灵测试”这样的一些概念，会继续学习书中的相关知识点；
3）这周继续对java进行了学习，本身java基础不太好，java里面的内容又很多，所以这个基础的学习不能间断。
4）初步接触了gitHub是一个面向开源及私有软件项目的托管平台，使用hexo搭建github的个人
域名博客，老师布置的这个任务让我实际进行操作，学习到了很多东西。以前没有接触过这个，感觉很新奇。
在搭建过程中我主要遇到了以下几个问题：
首先，在安装hexo时总是报错，后来发现是网络源的问题，导致镜像不通，解决方法是，把源换成淘宝源，
sudo npm install -g hexo –registry=https://registry.npm.taobao.org，然后将npm指令换成cnpm指令进行安装即可
其次：在绑定个人域名时不成功，原因是路径弄错，没有设置正确的路径；
最后：虽然我成功搭建了这个个人博客，但是这里面还有很多东西需要我去学习，我只是粗略的了解，还需要学习它更多的功能。
