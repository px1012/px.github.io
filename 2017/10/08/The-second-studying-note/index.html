The second studying note

发表于 2017-10-08 | 分类于 Thinking

第二周学习笔记
本周主要列举一些学习经验（hexo+github小白的学习之路，不正确的地方望指出）
1：如何更换主题（由默认的Landscape主题更换为我现在的主题next为例）
1）下载next主题
在hexo安装目录下用Git Bash here进入命令框输入以下命令下载next主题
git clone https://github.com/iissnan/hexo-theme-next themes/next
2)运用next主题
进入Hexo文件夹下打开配置文件_config.yml,找到theme字段，将其改为next
3）本地验证主题
hexo目录下，输入hexo s -debug进行调试，
此时本地浏览器输入http://localhost:4000/ 就能预览到该主题的默认效果
4）菜单项的设定
Themes目录下的config.yml配置文件，打开，其中menu节点，将几个注释都给去掉，
本地浏览器刷新看看博客首页菜单的变化，多试几次看看变化，改成自己喜欢的菜单
我是将所有注释都去掉了，代码如下
menu:
home: / || home
about: /about/ || user
tags: /tags/ || tags
categories: /categories/ || th
archives: /archives/ || archive
schedule: /schedule/ || calendar
sitemap: /sitemap.xml || sitemap
commonweal: /404/ || heartbeat
2：主页添加自己的头像（这里我遇到了一些问题）
在Hexo/source目录下新建文件夹uploads,将自己图片放入，在next/config.yml文件中搜索avatar，
将其前面注释去掉，更改图片路径为/uploads/px.jpg(自己的图片名，包括后缀都区分大小写)

Sidebar Avatar

in theme directory(source/images): /images/avatar.jpg

in site directory(source/uploads): /uploads/avatar.jpg

default : /images/default_avatar.jpg

avatar: /uploads/px.jpg
作者和简述
打开Hexo/config.yml，里面author改为自己的名称，description可填写自己喜欢的一句话，保存刷新即可看到效果。

Site

title: Note

language: zh-Hans//语言显示为中文
timezone:
注：若图片老是添加不成功，可尝试以下解决方法
1）avatar: /uploads/px.jpg 这句代码里冒号后要有空格
2）图片路径是否正确（可查看一下网页的源代码，看看路径文件名是否正确）
若需要修改.html文件，去仓库下载index.html文件解压修改后再上传到仓库
3）图片大小修改下可能太大
3：将更新部署到GitHub
在Hexo 目录下依次执行以下命令:
hexo clean
hexo generate
hexo server (若是想本地预览和执行该命令预览)
hexo deploy
